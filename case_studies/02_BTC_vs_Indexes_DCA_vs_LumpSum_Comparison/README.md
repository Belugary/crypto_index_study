# BTC 与指数投资策略对比研究

## 🎯 案例目标

通过对比分析回答关键投资问题：

- **单一资产 vs 多元化**: BTC 单独投资 vs 多元化指数投资的效果
- **投资方式**: 定投(DCA) vs 一次性投资(Lump Sum)的优劣
- **指数规模**: 不同市值规模指数(top2/5/10/30/50)的表现差异
- **长期效果**: 7 年多时间跨度的真实表现对比

## 📁 文件说明

### `收益率对比分析.py`

**一次性投资策略分析**

- 模拟在 2018 年 1 月 1 日投入资金
- 对比 BTC vs 各指数的总收益率
- 分析风险调整后收益
- 生成可视化图表和详细报告

### `定投收益率对比分析.py`

**定投策略分析**

- 模拟月度定额投资(每月$1000)
- 对比定投 vs 一次性投资效果
- 分析不同标的的定投表现
- 评估风险分散效果

## 🚀 运行方式

```bash
# 进入目录
cd case_studies/02_BTC_vs_Indexes_DCA_vs_LumpSum_Comparison/

# 运行一次性投资分析
python 收益率对比分析.py

# 运行定投策略分析

```

## 📊 分析维度

### 投资标的对比

- **BTC**: 比特币单一资产
- **Crypto2**: 前 2 名市值加权指数
- **Crypto5**: 前 5 名市值加权指数
- **Crypto10**: 前 10 名市值加权指数
- **Crypto30**: 前 30 名市值加权指数
- **Crypto50**: 前 50 名市值加权指数

### 关键指标

- **总收益率**: 整个投资期间的绝对收益
- **年化收益率**: 考虑时间因素的标准化收益
- **最大回撤**: 最大损失幅度
- **夏普比率**: 风险调整后收益指标
- **波动率**: 投资组合风险水平

### 投资策略

- **Lump Sum**: 起始日期投入全部资金
- **DCA**: 定期定额投资，分散时间风险

## 💡 研究假设

### 多元化效应

- **假设**: 多元化指数比单一 BTC 风险更低
- **验证**: 通过波动率和最大回撤对比

### 定投优势

- **假设**: 定投能够降低时机风险
- **验证**: 对比定投 vs 一次性投资的风险收益

### 集中度影响

- **假设**: 集中度越高收益越高，但风险也越大
- **验证**: 对比 top2 vs top50 的风险收益特征

## 📈 预期结果

基于历史数据和理论预期：

### 收益率排序(预期)

1. **BTC**: 最高收益，最高风险
2. **Crypto2**: 高收益，高风险
3. **Crypto5**: 较高收益，中等风险
4. **Crypto10**: 中等收益，中等风险
5. **Crypto30**: 较低收益，较低风险
6. **Crypto50**: 最低收益，最低风险

### 定投效果(预期)

- **熊市期间**: 定投优于一次性投资
- **牛市期间**: 一次性投资优于定投
- **整体期间**: 取决于具体的市场走势

## ⚠️ 重要提醒

### 数据局限性

- 基于历史数据，不代表未来表现
- 未考虑交易成本和税务影响
- 理想化的投资执行假设

- 加密货币投资风险极高
- 可能面临完全损失
- 需要根据个人情况理性投资
